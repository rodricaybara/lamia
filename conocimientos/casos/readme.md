# Casos de referencia

## Objetivo

Este directorio contiene casos reales correctamente resueltos.

Los casos sirven como referencia para interpretar la normativa y validar el comportamiento esperado de la aplicación.

Siempre que exista un caso de referencia aplicable, deberá utilizarse antes de consultar la documentación oficial.

---

# Contenido

Cada caso debe incluir:

- Identificador
- Estado
- Objetivo
- Contexto
- Datos relevantes
- Resultado esperado
- Reglas utilizadas
- Observaciones
- Fuente documental

---

# Buenas prácticas

Cada caso debe corresponder a una situación real.

No deben inventarse ejemplos.

Los casos no sustituyen a las reglas de negocio.

Su finalidad es mostrar cómo se aplican dichas reglas.

Cuando exista una resolución validada, deberá utilizarse como caso de referencia.
