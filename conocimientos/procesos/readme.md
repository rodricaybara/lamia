# Procesos

## Objetivo

Este directorio describe el funcionamiento de los procedimientos regulados por la normativa.

Un proceso representa una secuencia de actuaciones administrativas.

---

# Contenido

Cada proceso debe incluir:

- Objetivo
- Inicio
- Entradas
- Flujo del proceso
- Salidas
- Reglas aplicables
- Casos relacionados
- Fuente normativa

---

# Buenas prácticas

Los procesos describen el orden lógico de las actuaciones.

No contienen reglas detalladas.

Cada paso debe hacer referencia a las reglas correspondientes.

No describen la implementación informática.
