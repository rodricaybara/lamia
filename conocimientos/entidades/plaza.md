# Entidad: Plaza

## Objetivo

Representar una plaza objeto de un procedimiento de selección o contratación.

---

# Descripción

Una plaza define el puesto que se pretende cubrir mediante un procedimiento regulado por la normativa de la UPV/EHU.

Sus características determinan las reglas que deberán aplicarse durante el procedimiento de selección.

---

# Finalidad

Describir las características de la plaza sobre las que se aplican las reglas de negocio.

---

# Atributos principales

| Atributo | Descripción |
|----------|-------------|
| Código | Identificador de la plaza. |
| Tipo de plaza | Profesor Ayudante Doctor, Agregado, Titular, etc. |
| Área de conocimiento | Área asociada a la plaza. |
| Perfil lingüístico | Perfil lingüístico requerido, cuando proceda. |
| Departamento | Departamento responsable. |
| Instituto | Instituto universitario, cuando proceda. |

---

Las aplicables según el tipo de plaza.
