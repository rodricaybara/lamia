# Entidades

## Objetivo

Este directorio describe los conceptos utilizados por la normativa y por la aplicación.

Las entidades representan los elementos principales del dominio del problema.

Ejemplos:

- Comisión
- Plaza
- Candidato
- Convocatoria

---

# Contenido

Cada entidad debe describir:

- Descripción
- Finalidad
- Atributos
- Relaciones con otras entidades
- Reglas relacionadas
- Procesos relacionados
- Fuente normativa

---

# Buenas prácticas

Una entidad no contiene reglas de negocio.

Una entidad no describe procesos.

Debe limitarse a definir un concepto del dominio.
